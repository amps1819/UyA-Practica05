Tabla de las funciones/atributos más usuales

Funciones/Atributos Propósito
document.getElementById(elemento) Devuelve la identificación única del "elemento" seleccionado.
document.getElementsByTagName(tag) Devuelve un array con todos los elementos cuyo "tag" sea el seleccionado.
document.getElementsByClassName(clase) Devuelve un array con todos los elementos cuyo atributo "clase" sea el seleccionado.
elemento.innerHTML Es el atributo que recoge el contenido HTML del "elemento".
elemento.atributo Forma de acceder al "atributo" que pertenece al "elemento".
elemento.style.propiedad Forma de acceder a la "propiedad" CSS que pertenece al "elemento".
elemento.setAttribute(atributo, valor) Asigna el "valor" del "atributo" del "elemento"
document.createElement(elemento) Genera un nuevo "elemento" en la página.
elemento.removeChild(elemento)
elemento.appendChild(elemento)
elemento.replaceChild(antiguo, nuevo)
